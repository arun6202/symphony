Hard truths

The idea fails if semantics and evidence are treated as optional.

The hidden risks are not exotic. They are parent-change semantics, false metadata confidence, weak reconciliation, PII leakage, backfill complexity, and documentation drift.

Primary warning

CDC is not the hard part. Denormalized document correctness is the hard part.

Parent Changes

If customer, product, or category changes, do historical order documents update, preserve history, or trigger rebuild?

False Confidence

Draft OpenMetadata can look official before validation. Label, validate, and separate internal bundle from API payload.

Type Limits

Types prevent local shape errors. They do not prove foreign keys, secondary uniqueness, aggregates, or stream ordering.
