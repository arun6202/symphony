Domain contract compiler

Export models and portable business rules, not arbitrary programs.

F# remains the authoring source. A language-neutral DomainIR emits TypeScript, JavaScript, Kotlin, and Swift models, validators, codecs, fixtures, and selected pure rule implementations.

Golden rule

Do not transpile the whole program. Export a portable domain contract and prove it with shared golden tests.

TypeScript

Tagged unions, record types, validators, codecs, and exhaustive match helpers.

JavaScript

Runtime validators, JSDoc typedefs, plain constructors, and codec functions.

Kotlin

Data classes, sealed interfaces, value classes, kotlinx.serialization, typed validation.

Swift

Structs, enums with associated values, Codable codecs, Result-returning validators.
