Helios

The query, semantic, and product surface.

Helios turns generated Elasticsearch indices into safe typed queries, API workflows, semantic models, frontend controls, and quality evidence.

Library Direction

Layer a type-safe F# authoring DSL over the official Elasticsearch client. Keep Elastic's transport and admin APIs; replace fragile query construction with typed algebra.

Typed Fields

Keyword, text, text-with-keyword, date, numeric, bool, nested, edge-ngram, and search-as-you-type should drive legal operations.

Semantic Catalog

OSI-style artifacts provide labels, descriptions, relationships, metrics, UI hints, provider extensions, and synonyms.

Provider Compilers

Client predicates and aggregations lower into Elasticsearch, SQLite, or future engines without the UI knowing physical details.

Query DSL Shape

The lowering pass is the high-value part. It turns typed intent into correct Elasticsearch JSON.

let q =
    Shipments.Carrier =. "BlueDart"
    &&. Shipments.ShippedAt .between (DateMath.daysAgo 7, DateMath.today)
    &&. nested Shipments.Packages (fun p -> p.WeightKg >. 5.0<kg>)
    |> Query.where'
    |> Query.rankBy (Shipments.Notes |> phrase "fragile handling")
Risky ES behaviorHelios answer
Term query on analyzed textLower to keyword sub-field or make it unrepresentable.
Nested cross-object contaminationUse phantom nested scopes.
Date math as magic stringsUse a date-math DU and render at the edge.
Range on non-orderable fieldExpose range operators only on date/numeric handles.
Raw JSON escape hatchRequire a name, reason, and telemetry warning.
